What the task of React? Render view, update Dom only if needed
What is the Virtual Dom? A huge nested js object.
What happends when importing CSS in a JS file? Webpack includes the CSS into HTML.
What's the job of ReactDom? It puts the component into the DOM.
What does JSX? It uses createReact() under the hood to build React components.
Why components starts with a capital letter? To distinguish components from HTML elements.
What is '<>'? Shortform for '<React.Fragment>''
Why classnames are sometimes capitalized? To show that the classname belongs to a component.
How can you access a DOM element? ReactDOM.findDOMNode()
How can I set checked? checked={bol}
What is the job of the React package? Building and managing the virtual DOM, a HTML-tree like strutured object
What is the job of the ReactDom package? Building the HTML tree (DOM), it's the glue between React and the DOM
