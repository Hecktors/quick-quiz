What is the MERN Stack? It consists of the following technologies: MongoDB, Express, React, Node.js.
What is Node.js? It's a run-time environment that allows you to execute JavaScript code outside of the browser.
What is Express? A web application framework for Node.js. It allows us to easily create a web server.
What is MongoDB? A document-based open- ource database.
What is Mongoose? A simple, schema-based solution to model application data. It makes it easier to use MongoDB in Node.js.
Database Concepts (Tablar (Relational) - MongoDB) Database - Database, Table - Collection, Row - Document, Index - Index, Join - $lookup, Foreign Key - Reference
What is cors? cross-origin resourse sharing - Allow ajax requests to skip the same origin policy and acces resources from remote host. To access something outside from our server from our server
What is dotenv for? It loads environment variables into out process env. To store env var in a file.
