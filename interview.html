What is JSX? JSX is syntatic sugar for the React.createElement function
Wie überreiche ich einen Wert von Parent zu Child? Der Wert wird als Props übergeben.
Wie übergebe ich einen Wert von Child zu Parent? Es muss dem Child eine im Parent erstellte Funktion übergeben werden, die vom Child mit dem Wert als Parameter aufgerufen wird.
Was ist Prop drilling? Eine Komponente empfängt einen Wert vom Parent und gibt es, ohne es selber zu nutzen, an das Kind weiter.
Was sind die Cons von ContextAPI Wenn ein Wert geändert wird, werden alle Komponenten neu gerendert. Deshalb bei Etwas benutzen, was mögl. viele Komoponente betrifft( Theming)
Was ist ein Fragment? Eine Möglichkeit mehrere child elementen zum umschließen, ohne ein zus. div benutzen zu müssen.
Wann muss man Klassen Komponenten benutzen? Wenn man lifecyle Methoden nutzen muss für die es noch keine Hooks Alternative gibt.
Was ist eine HOC? Eine Komponente, die einen andere Komponente entgegennimmt und diese modifiziert weitergibt.
Was sind controlled Components? Input Komponente, wie input, textarea oder select, dessen Werte von React kontrolliert werden.
